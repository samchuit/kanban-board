# 📋 OpenClaw 任务看板

OpenClaw 个人助手的任务管理与状态追踪看板。

## ✨ 功能特性

### 核心功能
- **四列看板**: 待办 → 进行中 → 待审核 → 已完成
- **拖拽操作**: 轻松在列之间移动任务
- **任务详情**: 标题、描述、优先级、标签、截止日期
- **实时统计**: 总任务数、已完成数、进行中数、完成率
- **搜索过滤**: 快速查找任务
- **数据持久化**: LocalStorage 自动保存

### 优先级系统
- 🔴 高优先级 - 紧急任务
- 🟡 中优先级 - 普通任务
- 🟢 低优先级 - 可以在空闲时处理

## 🚀 快速开始

### 方法1: 使用 Python（推荐）
```bash
cd kanban-project
python3 -m http.server 8080
```
然后打开浏览器访问: http://localhost:8080

### 方法2: 直接打开
```bash
cd kanban-project
open index.html
```

### 方法3: 使用 Node.js
```bash
cd kanban-project
npm install -g serve
serve .
```

## 💾 数据存储

所有数据保存在浏览器 LocalStorage 中：
- 数据会自动保存
- 刷新页面后数据不会丢失
- 可通过"导出"按钮备份数据

## 🎮 使用说明

### 添加任务
1. 点击 "+ 添加任务" 按钮
2. 填写任务信息（标题必填）
3. 点击"保存任务"

### 编辑任务
- 点击任意任务卡片即可编辑

### 移动任务
- 拖拽任务卡片到其他列
- 状态会自动更新

### 删除任务
- 编辑任务时点击删除按钮

### 搜索任务
- 在顶部搜索框输入关键词
- 实时过滤任务列表

### 导出数据
- 点击"导出"按钮下载 JSON 备份
